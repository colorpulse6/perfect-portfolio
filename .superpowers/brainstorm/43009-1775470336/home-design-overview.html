Components

  • ParticleField — R3F instanced mesh, 200-300 points, cursor repulsion physics, bloom post-processing
  • FloatingArtifact — R3F sprite, drifts across scene, glows on proximity, bursts into particles on click, triggers console
  • ArtifactManager — Manages artifact pool: spawns from edges, staggers timing, handles respawn after burst
  • TerminalConsole — DOM overlay, GSAP slide-in, character-by-character typing, command input with recognized commands
  • HomeScene — R3F Canvas wrapper, composes ParticleField + ArtifactManager, handles pointer events

Mobile Behavior

  • Particles: Reduced to ~100, no bloom
  • Artifacts: Same behavior, touch = cursor
  • Console: Full-width, display-only (no keyboard input), dismiss button
